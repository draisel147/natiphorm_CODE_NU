--------------- กฎการตั้งชื่อตัวแปร ---------------
1.ตัวแปรจะประกอบด้วยตัวเลข ตัวอักษร เครื่องหมาย
2.ตัวอักษรตัวแรก*ห้าม*ขึ้นต้นด้วยตัวเลขหรือสัญลักษณ์พิเศษ 
ยกเว้น _ และ $ 
3.ห้ามซ้ำกับคำสงวน final for if etc.
4.Case Sensitive ตัวพิมเล็กตัวพิมใหญ่มีความหมายต่างกัน(คนละตัวกัน)
void main(){
    String name = 'ney';
    String Name = 'natiphorms';
    String NAME = 'dart basic';
}

ทั้งหมดนี้จะเป็นคนละตัวกัน Case Sensitive
